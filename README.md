# Emometer

## Installation

```shell
git clone git@github.com:nordichackday/votingapp.git
cd votingapp
npm install
```

Add video files in the `public/video` directory.

## Build

```shell
npm run build
```

## Watch

Start the server and build daemon:

```shell
npm run watch
```

Then visit the application at [http://localhost:8081/](http://localhost:8081/)
